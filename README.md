# 金额大小写转换工具

一个纯静态网页工具，用于把数字金额转换为：

- 人民币大写
- 英文金额表达
- 英文支票写法

## 本地预览

直接用浏览器打开 `index.html` 即可。

## 部署方式

### Vercel

1. 新建一个 Vercel 项目。
2. 选择本文件夹作为项目根目录。
3. Framework Preset 选择 `Other`。
4. Build Command 留空。
5. Output Directory 留空或填写 `.`。
6. 点击 Deploy。

### Netlify

1. 新建一个 Netlify 站点。
2. 选择本文件夹作为项目根目录，或直接上传压缩包。
3. Build command 留空。
4. Publish directory 填写 `.`。
5. 点击 Deploy。

### GitHub Pages

1. 将本文件夹提交到 GitHub 仓库。
2. 在仓库 Settings -> Pages 中选择部署分支。
3. Source 选择仓库根目录。
4. 保存后等待 GitHub 生成公开网址。
